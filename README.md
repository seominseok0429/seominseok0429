### 안녕하세요! 인공지능 연구원 서민석 입니다.

----

저는 인공지능 모델이 실제로 제품화되고 응용되는것에 관심이 있으며, 주 연구 분야로는 현실에서 발생할 수 있는 **damaged images에 강인한 인공지능 모델 개발**, 현실에서 구하기 어려운(재난상황, 군사데이터)데이터를 가상으로 생성하고 **Domain Adaptation**을 통하여 극복하는 연구 등이 있습니다.
